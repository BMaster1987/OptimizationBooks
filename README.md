# Optimization Library and Python Materials / Biblioteca de Otimização e Materiais em Python

## Português

Este repositório reúne **livros, slides e notebooks** relacionados a otimização matemática, otimização convexa, otimização numérica, validação de sistemas e tomada de decisão.

O objetivo é centralizar, em um único lugar, materiais de estudo e apoio computacional para consulta acadêmica, aprendizado e implementação de algoritmos.

## Conteúdo do repositório

### 1. Livro principal
